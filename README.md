# UserBehaviorAnalysisSystem
#### 项目介绍：
>这是一个基于vue的简单新闻网站和该网站对应的用户行为分析系统新闻网部分包括首页、登录注册页、新闻详情页；用户行为分析系统部分主要通过分析该网站的特定时间段的登录数、注册数、访问数、广告访问数等行为等。
#### 运行：
启动前端

```
npm run dev
```
>news_backend/mysql/news.sql 数据库初始化的sql文件

启动后端
```
npm run server
```
